Esta guía te lleva paso a paso. Cada paso tiene un tiempo estimado. Al final tendrás todo funcionando.

📤 Paso 1: Subir la Plantilla a Google Sheets ⏱️ 3 min

Convierte tu plantilla Excel en un Google Sheet que será la base de datos central.

1.1 — Abrir Google Drive

Ve a drive.google.com y navega a la carpeta donde quieres guardar el archivo.

1.2 — Subir el archivo Excel

Haz clic en + Nuevo → Subir archivo y selecciona Plantilla_Dashboard_Autos_Zurich.xlsx desde tu computador.

1.3 — Convertir a Google Sheets

Una vez subido, abre el archivo. Arriba aparecerá un banner que dice "Abrir con Google Sheets". Haz clic ahí. Luego ve a Archivo → Guardar como Google Sheets. Esto crea una copia nativa de Google que permite usar Apps Script.

1.4 — Verificar las hojas

Confirma que se ven las 6 pestañas abajo: Gestión Diaria, Tipificación Detallada, Plan de Acción, Inversión Pauta, Catálogos, Instrucciones.

Si alguna fórmula se ve rara, no te preocupes. El formato de Google Sheets puede variar un poco del Excel, pero los datos estarán bien.

⚡ Paso 2: Instalar el Script de Automatización ⏱️ 5 min

Este script es el cerebro de la automatización. Crea formularios, procesa datos y genera resúmenes.

2.1 — Abrir el editor de Apps Script

En tu Google Sheet, ve al menú: Extensiones → Apps Script. Se abrirá una nueva pestaña con el editor de código.

2.2 — Pegar el código

En el editor verás un archivo Código.gs con algo de código por defecto. Selecciona todo (Ctrl+A) y bórralo. Luego abre el archivo Apps_Script_Automatizacion.js con un editor de texto (Notepad, VS Code, o simplemente haz clic derecho → Abrir con → Bloc de notas), copia TODO el contenido y pégalo en el editor de Apps Script.

2.3 — Guardar

Haz clic en el ícono de disco 💾 o presiona Ctrl+S para guardar. El nombre del proyecto puede ser cualquier cosa, por ejemplo "Automatización Zurich Autos".

2.4 — Ejecutar la configuración inicial

En la barra de funciones arriba del editor, busca el selector que dice configurarInicial. Si no aparece, selecciónala del menú desplegable. Luego haz clic en el botón ▶ Ejecutar.

2.5 — Autorizar permisos

Google te pedirá que autorices el script. Sigue estos pasos:

1. Clic en Revisar permisos
2. Selecciona tu cuenta de Google
3. Si dice "Esta app no está verificada", haz clic en Avanzado → Ir a Automatización Zurich Autos (no seguro)
4. Clic en Permitir

2.6 — Verificar que funcionó

Después de ejecutar verás un cuadro de confirmación que dice "✅ Configuración Completa". Vuelve a tu Google Sheet y deberías ver:

  • ✓ Un nuevo menú "🚗 Zurich Autos" en la barra de menú
  • ✓ Una nueva hoja "Resumen Automático" con fórmulas en vivo
  • ✓ Una nueva hoja "Importar Sistema" con instrucciones de importación
  • ✓ Un Google Form vinculado (verás un diálogo con el link)
El paso de autorización es normal y seguro. Google pide estos permisos porque el script necesita acceder a tu Sheet y crear el formulario. Es código tuyo, no de un tercero.
Guarda el link del formulario que aparece en el diálogo. Lo vas a necesitar para compartirlo con tus agentes por WhatsApp.

📱 Paso 3: Compartir el Formulario con los Agentes ⏱️ 3 min

El formulario permite que tus agentes reporten su gestión diaria desde el celular.

3.1 — Obtener el link del formulario

En tu Google Sheet, ve al menú 🚗 Zurich Autos → 📱 Ver Link del Formulario. Aparecerá un diálogo con el link. Cópialo.

3.2 — Compartir por WhatsApp

Envía el link al grupo de WhatsApp de tus agentes con un mensaje como:

🚗 Equipo, a partir de hoy vamos a reportar la gestión diaria por este formulario. Es rápido (menos de 1 minuto) y nos ayuda a tener los datos en tiempo real.

📋 Link: [pega el link aquí]

Llénenlo al final de cada jornada con los números del día. ¡Gracias! 💪

3.3 — Qué llena cada agente

El formulario le pregunta al agente:

  • 📋 Su nombre (lista desplegable con las 7 agentes)
  • 📅 Fecha de reporte
  • 📡 Fuente de los leads (Pauta, Pauta Max, Inbound, etc.)
  • 📊 Números: registros, contactados, cotizados, ventas, prima
  • 📞 Canal (Llamada o WhatsApp)
  • ❌ Razones de no cotización (si aplica)
  • 📝 Observaciones (opcional)

3.4 — Qué pasa cuando envían el formulario

Cada vez que un agente envía una respuesta, el script automáticamente:

  • ✓ Toma la respuesta y la formatea correctamente
  • ✓ La escribe en la hoja "Gestión Diaria" en una nueva fila
  • ✓ Calcula automáticamente: Aptos, % Contactabilidad, % Cierre
  • ✓ El "Resumen Automático" se actualiza en tiempo real
Los agentes pueden guardar el link como favorito en el navegador del celular, o incluso agregarlo a la pantalla de inicio para acceso rápido.

💾 Paso 4: Importar Datos del Sistema Interno ⏱️ 5 min

Para cargar datos masivos desde tu sistema interno, usa la función de importación CSV.

4.1 — Exportar del sistema interno

En tu sistema, exporta los datos de gestión del día como CSV o Excel. Necesitas al menos estas columnas:

Agente | Fecha | Fuente | Registros | Contactados | Cotizados | Ventas | Prima

4.2 — Pegar en la hoja de importación

Ve a la hoja Importar Sistema en tu Google Sheet. Desde la fila 11 hacia abajo, pega los datos del sistema. El script espera las columnas en este orden:

Columna A: Agente | B: Fecha | C: Fuente (como venga del sistema) | D: Registros | E: Contactados | F: Cotizados | G: Ventas | H: Prima

4.3 — Ejecutar la importación

Ve al menú 🚗 Zurich Autos → 📥 Importar datos del sistema. El script:

  • ✓ Lee los datos pegados
  • ✓ Mapea las fuentes del sistema a los nombres estándar (ej: "wa pauta" → "WhatsApp Pauta")
  • ✓ Escribe todo en "Gestión Diaria" con el formato correcto
  • ✓ Limpia la hoja de importación para la próxima vez
El script reconoce variaciones de nombres de fuente. Por ejemplo: "wa pauta", "whatsapp pauta", "WA_PAUTA" se convierten en "WhatsApp Pauta".

🔗 Paso 5: Conectar el Dashboard con Google Sheets ⏱️ 5 min

El último paso: haz que tu Dashboard lea datos en vivo desde Google Sheets.

5.1 — Publicar el Google Sheet en la web

En tu Google Sheet, ve a Archivo → Compartir → Publicar en la web.

1. En "Enlace", selecciona Todo el documento
2. En formato, selecciona Valores separados por comas (.csv)
3. Haz clic en Publicar
4. Confirma en el diálogo que aparece

5.2 — Copiar el ID del Sheet

El ID está en la URL de tu Google Sheet. Mira la barra de direcciones del navegador:

https://docs.google.com/spreadsheets/d/ESTE_ES_EL_ID/edit

Copia solo la parte amarilla (es un texto largo con letras y números).

5.3 — Configurar el Dashboard

Abre el archivo Dashboard_Autos_Zurich_2026.html en tu navegador (doble clic). En la esquina superior derecha verás un ícono de engranaje ⚙️. Haz clic para abrir el panel de configuración.

1. Pega el ID del Sheet en el campo
2. Haz clic en Conectar y Sincronizar
3. Si todo está bien, verás "✅ Sincronizado" y los datos se cargarán

5.4 — Uso diario

A partir de ahora, cada vez que abras el Dashboard hará lo siguiente:

  • 🔄 Recuerda el ID del Sheet (guardado en tu navegador)
  • 📊 Lee los datos más recientes de Google Sheets
  • 📈 Actualiza todos los gráficos y KPIs automáticamente
  • ✅ Sincroniza el plan de acción
El Sheet debe estar publicado en la web para que el Dashboard pueda leerlo. Si lo despublicas, el Dashboard usará los datos que tenga en caché.
Puedes compartir el archivo HTML con cualquier persona del equipo. Cada quien lo abre en su navegador y configura el mismo ID del Sheet. Todos ven los mismos datos en tiempo real.
